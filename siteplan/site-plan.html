Individual Project Site Plan

Finance Tracker

Lewis Karume

1. Site Name

Project Name: Finance Tracker

Reason: I chose this name because it is short, memorable, and clearly describes the core functionality of the application which is tracking all finances to maintain financial health and see savings in real time.

2. Site Purpose

Finance Tracker is a personal finance tool designed to help users track their income and expenses. The site provides a dashboard to log daily transactions, visualizes spending habits through dynamic charts, and allows users to set and track monthly budget limits.

3. Scenarios

The following questions represent the needs of the target audience:

  • "I just bought groceries and want to quickly record the cost. How do I add a transaction?"
  • "I feel like I'm spending too much on eating out. How can I see how much I spent on restaurants last month?"
  • "I want to set a monthly budget for entertainment expenses. How do I do that?"
  • "I want to see a visual breakdown of my spending categories. How can I create that?"
  • "I want to export my transaction history to a CSV file. How can I do that?"

5. Typography

The site will use Google Fonts for clear readability.

  • Headings: Roboto (Used for page titles and section headers)
  • Body: Open Sans (Used for transaction lists and general text)
